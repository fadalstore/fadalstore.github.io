💻 Casharka 2aad – Heerka 1aad

Deegaanka Shaqada — Setup Python & Terminal

⏱ ~50 Daqiiqo📖 Heer: Bilow🌍 Af-Soomaali

Maxaad Baranaysaa Casharkaan?

Casharkaan wuxuu kugu barin doonaa sida aad ugu rakibto Python qalabkaaga iyo sida aad u bilaabayso isticmaalka Terminal-ka. Waxaad qori doontaa koodhaadii ugu horeeyay!

📌 Waxaad baranaysaa: Sida loo rakibo Python (Windows, Mac, Linux, iyo Telefoon), sida loo furto Terminal-ka, amarka aasaasiga ah, iyo sida loo qoro "Hello World" koodhaaga ugu horreeya.

1. Dooro Qalab Aad Isticmaalayso

Python waxaad ku raadin kartaa qalabo badan oo kala duwan. Dooro mid ka mid ah:

💻 Option A: Kombuyuutar (Ugu Fiican)

Haddaad leedahay kombuyuutar (Windows, Mac, ama Linux), tani waa habka ugu fiican ee aad ku baranayso Python.

📱 Option B: iPhone — App iSH

Haddaad leedahay iPhone, waxaad isticmaali kartaa app-ka iSH Shell oo bilaash ah App Store-ka. Wuxuu kuu siin doonaa deegaan Linux ah oo aad ku codsan karto Python.

📱 Option C: Android — Termux

Android-ka isticmaalayaasha, app-ka Termux waa ikhtiyaar fiican. Waxaad ka dejisan kartaa Play Store-ka ama F-Droid.

2. Rakibidda Python — Tallaabo-tallaabo

🪟 Windows

1Tag python.org/downloads

2Riix "Download Python 3.x.x" (nooca ugu dambeeyay)

3Muhiim: Xusuus "Add Python to PATH" oo hoos ku calaamadee ✓

4Dhamee rakibidda oo dhagsii "Install Now"

5Xaqiijiso: Fur Command Prompt oo qor: python --version

🍎 macOS

1Fur Terminal (Cmd + Space → qor "Terminal")

2Haddaad Homebrew leedahay: brew install python3

3Ama tag python.org oo dejiso installer-ka

4Xaqiijiso: python3 --version

🐧 Linux (Ubuntu/Debian)

1Fur Terminal

2Cusbooneysii: sudo apt update

3Rakib: sudo apt install python3 python3-pip

4Xaqiijiso: python3 --version

📱 iPhone (iSH)

1Dejiso iSH Shell App Store-ka

2Fur iSH oo qor: apk add python3

3Xaqiijiso: python3 --version

📱 Android (Termux)

1Dejiso Termux

2Qor: pkg update && pkg install python

3Xaqiijiso: python --version

3. Barasho Terminal-ka — Amarka Aasaasiga ah

Terminal-ka (ama Command Line) waa aalad muhiim ah ee programmer-yada. Waxay kuu oggoshahay inaad ammaraan kompyuutarka si toos ah, aan isticmaalin interface-ka garafiga ah.

Amarka Aasaasiga ah ee Terminal-ka

# Arag meeshaan aad joogto (Print Working Directory)
pwd

# Liiska faylasha & galalka
ls

# Gal galka (Change Directory)
cd Documents

# Ka soo bax galka
cd ..

# Samee galka cusub
mkdir MashaariictaydaPython

# Samee faylka cusub
touch casharka1.py

# Qor qoraalka faylka
echo "print('Hello')" > casharka1.py

💡 Talo: Tab key-ga riix si aad u dhamaystirto amarka si dhakhso ah. Tusaale: qor "Doc" ka dibna riix Tab — kompyuutarku wuxuu dhammaystiri doonaa "Documents".

4. Qor Koodhaadii Ugu Horeeyay!

Hadda waxaad diyaar u tahay inaad qorto koodhaadii ugu horeeyay. Python-ka waxaad isticmaali kartaa laba hab:

Hab 1: Interactive Mode (REPL)

Fur Terminal-ka oo qor python3 (ama python Windows-ka), ka dibna ku qor:

>>> print("Ku soo dhowow Fadal Store!")
Ku soo dhowow Fadal Store!

>>> print("Magacaygu waa Python Baranaha!")
Magacaygu waa Python Baranaha!

Si aad uga baxdo interactive mode-ka: qor exit() ama riix Ctrl+D.

Hab 2: Qoro Faylka .py

Tani waa habka xirfadlayaasha joogta ah isticmaalaan. Samee faylo Python ah:

# 1. Samee faylka
# Terminal-ka ku qor:
# touch hello.py

# 2. Fur faylka editor-ka leh (nano ama notepad)
# Terminal: nano hello.py

# 3. Qor koodhkan faylka dhexdiisa:
print("Ku soo dhowow Fadal Store!")
print("Casharka 2aad - Setup Python")
print("Waxaan baranayaa Python Af-Soomaali")

Ka dibna run-garee:

python3 hello.py

Natiijada:

Ku soo dhowow Fadal Store!
Casharka 2aad - Setup Python
Waxaan baranayaa Python Af-Soomaali

5. VS Code — Editor-ka Ugu Fiican (Ikhtiyaari)

Haddaad kombuyuutar leedahay, waxaan kugula talineynaa inaad dejisato Visual Studio Code (VS Code) — waa bilaash oo uu Microsoft sameeyay. Wuxuu kaa caawinaa in koodhaagu yahay mid kala sooc leh oo nadiif ah.

  1. Tag code.visualstudio.com
  2. Dejiso noocaaga system-ka
  3. Ku rakib extension-ka "Python" (Microsoft)
  4. Furitaan faylka .py, dhijitaal F5 si aad u socodsiso

⚠️ Khalad Caadi ah: Haddaad hesho "python: command not found", isku day python3 beddelkiis. Windows-ka, xaqiijiso in PATH-ka si sax ah loo dejiyay marka rakibidda.
